# NewsApp

This project is a sample news application project that I did during my internship. News data is taken from (https://newsapi.org/) site.
<br>
<br>

-------- Used in the project----------
* Retrofit
* Recylerview
* Firebase 
* Google and Facebook login
* Deeplink
* MVVM
* LiveData
